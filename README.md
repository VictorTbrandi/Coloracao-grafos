# Coloração de Grafos - Welsh-Powell com Fila

Projeto Java para trabalho de Pesquisa Operacional utilizando coloração de grafos.

## Como executar no IntelliJ

1. Abra a pasta do projeto no IntelliJ.
2. Aguarde o Maven carregar.
3. Execute a classe `br.edu.grafos.Main`.
4. Digite o grafo manualmente no console seguindo as instruções exibidas.

## Como executar pelo terminal

```bash
mvn clean compile exec:java
```

O programa monta a lista de adjacência, executa o Algoritmo Maior Primeiro com fila, mostra os estados da fila e apresenta a matriz final das cores.
